# NoteScribe 1.0.0

- Standalone macOS transcription app (Parakeet V2) with bundled ASR and VAD models when available; falls back to FluidAudio download if missing.
- Menu-bar dictation (hotkey/push-to-talk), file transcription, and a scratchpad home tab.
- Voice Activity Detection optional per-mode (live vs. file) with defaults configurable in Settings.
- Offline operation when models are bundled and staged on first launch.
