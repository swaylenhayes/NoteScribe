# NoteScribe 1.0.0

- Standalone macOS transcription app (Parakeet V3) with bundled ASR and VAD models; no additional downloads required after install.
- Menu-bar dictation (hotkey/push-to-talk), file transcription, and a scratchpad home tab.
- Voice Activity Detection optional per-mode (live vs. file) with defaults configurable in Settings.
- Offline operation: all models packaged inside the app bundle and staged on first launch.
